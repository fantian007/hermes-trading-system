## CAT-001 双均线交叉策略 (Dual MA Crossover)
- **类别**: 趋势跟踪
- **核心**: 短均线上穿长均线→BUY；下穿→SELL
- **入场条件**: MA5/MA10 上穿 MA20/MA50 金叉确认（站稳2根K线）+ 成交量放大
- **出场条件**: 反向交叉 或 价格跌破MA20
- **止损**: ATR(14) × 1.5
- **适用周期**: 日线/4H
- **核心指标**: MA5, MA10, MA20, MA50, ATR
- **胜率预估**: 35%-45%（盈亏比通常>2:1）
- **注意事项**: 震荡市假信号多，需配合趋势过滤
- **参考**: Investopedia Moving Average Crossover Strategy

## CAT-002 MACD 背离策略 (MACD Divergence)
- **类别**: 反转/动量
- **核心**: 价格创新高/新低但MACD指标未同步，预示趋势衰竭
- **入场条件**: 顶背离（价格新高+MACD柱未新高）→SELL；底背离（价格新低+MACD柱未新低）→BUY
- **出场条件**: MACD回归零轴 或 突破趋势线
- **止损**: 前高/前低 + ATR缓冲
- **适用周期**: 4H/日线/周线（越大越可靠）
- **核心指标**: MACD(12,26,9)
- **胜率预估**: 40%-60%
- **注意事项**: 背离可连续多次出现才反转，需等确认K线
- **参考**: TradingView MACD Divergence Guide

## CAT-003 凯尔特纳通道突破策略 (Keltner Channel)
- **类别**: 趋势/波动率
- **核心**: 基于ATR动态通道，突破捕捉趋势
- **入场条件**: 收盘突破上轨(EMA20+ATR×2)→BUY；跌破下轨→SELL
- **出场条件**: 价格回到中轨(EMA20) 或 移动止损ATR×2
- **止损**: 入场K线低点下方1 ATR
- **适用周期**: 1H/4H/日线
- **核心指标**: 凯尔特纳通道, ATR, 成交量
- **胜率预估**: 40%-55%
- **注意事项**: 基于ATR，更适合趋势市；与布林带不同
- **参考**: Keltner, Chester *How to Make Money in Commodities*

## CAT-004 抛物线转向策略 (Parabolic SAR)
- **类别**: 趋势跟踪
- **核心**: SAR点自动跟踪价格，提供动态止损和反转信号
- **入场条件**: SAR从价上翻到价下→BUY；从价下翻到价上→SELL
- **出场条件**: SAR再次翻转（SAR本身即动态出场信号）
- **止损**: SAR点本身
- **适用周期**: 日线/4H
- **核心指标**: Parabolic SAR(0.02, 0.2)
- **胜率预估**: 35%-45%
- **注意事项**: 震荡市SAR频繁翻转导致连续亏损
- **参考**: Wilder, J. Welles *New Concepts in Technical Trading Systems*

## CAT-005 VWAP成交量加权策略
- **类别**: 日内动量
- **核心**: VWAP是机构公允价，价格围绕VWAP波动
- **入场条件**: 价格上穿VWAP+放量→BUY；下穿VWAP+放量→SELL
- **出场条件**: 偏离VWAP>2个标准差 或 回到VWAP另一侧
- **止损**: VWAP下方/上方0.5%~1%
- **适用周期**: 1min/5min/15min（日内专用）
- **核心指标**: VWAP, 成交量, VWAP标准差通道
- **胜率预估**: 55%-65%
- **注意事项**: 仅日内有效，隔夜数据需重置
- **参考**: QuantConnect VWAP Trading Strategy

## CAT-006 成交量分布策略 (Volume Profile)
- **类别**: 微观结构/订单流
- **核心**: 分析成交量在价格上的分布，找价值区间(VA)和高流量节点(POC)
- **入场条件**: 突破VAH+成交量确认→BUY；跌破VAL+成交量确认→SELL；POC附近看反转信号
- **出场条件**: 达到下一个高流量节点 或 成交量衰减/背离
- **止损**: 跌破/突破关键节点（低流量区域）
- **适用周期**: 日内5min~1H
- **核心指标**: 成交量分布(VP), POC, 价值区间(VA)
- **胜率预估**: 50%-65%
- **注意事项**: 需专业软件；学习曲线陡峭
- **参考**: Steidlmayer, Peter *Markets and Market Logic*

## CAT-007 配对套利策略 (Pairs Trading)
- **类别**: 套利/市场中性
- **核心**: 两只高相关股票的价差偏离均值时交易回归
- **入场条件**: Z-score偏离>2σ。Z<-2→做多价差(做多弱势做空强势)；Z>2→做空价差
- **出场条件**: Z-score回归≈0
- **止损**: Z-score达到3 或 价差继续扩大N%
- **适用周期**: 日线/1H
- **核心指标**: 协整检验, Z-score, 相关系数, 半衰期
- **胜率预估**: 60%-75%
- **注意事项**: 相关关系可能突变，需定期检验协整
- **参考**: Vidyamurthy *Pairs Trading: Quantitative Methods*

## CAT-008 三屏交易系统 (Triple Screen)
- **类别**: 多时间框架/趋势
- **核心**: Alexander Elder，三个时间框架过滤信号
- **入场条件**: ①周线确认趋势方向(MACD柱方向)→②日线等回调至关键均线(EMA13)→③4H/1H找入场点(突破/RSI)
- **出场条件**: 移动止损(ATR) 或 趋势反转信号
- **止损**: 入场K线外1.5 ATR
- **适用周期**: 周线/日线/4H（三重时间框架）
- **核心指标**: MACD, EMA13, ATR, RSI
- **胜率预估**: 50%-65%
- **注意事项**: 需三屏同步，不适合快速交易
- **参考**: Elder, Alexander *Trading for a Living*

## CAT-009 截面动量策略 (Cross-Sectional Momentum)
- **类别**: 动量/量化
- **核心**: Jegadeesh&Titman：过去表现好的股票继续表现好
- **入场条件**: 过去6-12个月涨幅前10%~20%的赢家组合→BUY；跌幅前10%~20%的输家组合→SELL
- **出场条件**: 定期再平衡（每季度/每半年） 或 动量衰减信号
- **止损**: 组合层面最大回撤止损
- **适用周期**: 月线（中长线，持有3-12个月）
- **核心指标**: 过去6-12个月累计收益率
- **胜率预估**: 70%-85%（学术回测）
- **注意事项**: 金融危机中动量策略暴跌；1月份效应
- **参考**: Jegadeesh and Titman (1993)

## CAT-010 长期反转策略 (Long-term Contrarian)
- **类别**: 反转/价值
- **核心**: DeBondt&Thaler：过度反应的股票在未来反转
- **入场条件**: 过去3-5年累计跌幅前10%~20%→BUY（输家组合）；涨幅前10%~20%→SELL（赢家组合）
- **出场条件**: 组合回归正常 或 基本面改善信号消失
- **止损**: 组合-20%回撤
- **适用周期**: 月线/年线（超长线，持有3-5年）
- **核心指标**: 长期累计收益率, PB/PE
- **胜率预估**: 60%-75%
- **注意事项**: 需要极大耐心；输家可能真的基本面恶化（价值陷阱）
- **参考**: DeBondt and Thaler (1985)

## CAT-011 ADX趋势强度策略
- **类别**: 趋势/过滤
- **核心**: ADX衡量趋势强度，+DI/-DI提供方向
- **入场条件**: ADX>25(趋势确认) + +DI>-DI且ADX上升→BUY；-DI>+DI且ADX上升→SELL
- **出场条件**: ADX<25(趋势消失) 或 +DI/-DI交叉反转
- **止损**: ATR×2 或 DI反转
- **适用周期**: 1H/4H/日线
- **核心指标**: ADX(14), +DI(14), -DI(14)
- **胜率预估**: 40%-55%
- **注意事项**: ADX上升时才交易；ADX下降不宜入场
- **参考**: Wilder, J. Welles *New Concepts in Technical Trading Systems*

## CAT-012 三重指标共振策略 (Bollinger+RSI+Stochastic)
- **类别**: 震荡/多指标共振
- **核心**: 三个指标同时超买/超卖提高准确率
- **入场条件**: 布林下轨+RSI<30+Stoch%K<20金叉→BUY；布林上轨+RSI>70+Stoch%K>80死叉→SELL
- **出场条件**: 布林中轨(MA20) 或 Stochastic反向交叉
- **止损**: 布林外侧1ATR
- **适用周期**: 4H/日线
- **核心指标**: 布林带(20,2), RSI(14), Stochastic(14,3,3)
- **胜率预估**: 55%-70%
- **注意事项**: 强趋势中三重共振也失败，结合ADX过滤
- **参考**: Triple Confirmation Trading Strategy

## CAT-013 PEAD盈利公告漂移策略
- **类别**: 事件驱动/量化
- **核心**: 盈利超预期的股票公告后持续上涨数周至数月
- **入场条件**: EPS超预期>5%-10%的股票，公告后1-2交易日入场做多；低于预期→做空
- **出场条件**: 持有至下季度公告前 或 动量反转信号
- **止损**: 入场价下方5%-8%
- **适用周期**: 周线/月线（持有数周-数月）
- **核心指标**: 盈利惊喜(Earnings Surprise), Zacks Rank
- **胜率预估**: 60%-70%
- **注意事项**: 小市值漂移效应更显著；需考虑整体情绪
- **参考**: Bernard and Thomas (1989) Post-Earnings-Announcement Drift

## CAT-014 开盘区间突破策略 (Opening Range Breakout)
- **类别**: 日内动量
- **核心**: 开盘前30分钟形成区间(ORB)，突破后当天趋势延续
- **入场条件**: 突破开盘区间上沿+成交量放大→BUY；跌破下沿+成交量放大→SELL
- **出场条件**: 收盘前平仓（避免隔夜风险） 或 反向突破区间
- **止损**: 开盘区间另一侧
- **适用周期**: 5min/15min（日内专用）
- **核心指标**: 开盘区间(OR), 成交量, 前日收盘价
- **胜率预估**: 55%-65%
- **注意事项**: 盘整日OR失败率高；跳空需调整区间定义
- **参考**: Opening Range Breakout - Linda Raschke

## CAT-015 网格交易策略 (Grid Trading)
- **类别**: 震荡/自动化
- **核心**: 在价格区间内设多层挂单，低买高卖自动循环
- **入场条件**: 设定价格区间，等间距N层网格，每层同时挂买入和卖出单
- **出场条件**: 价格突破区间上沿/下沿停止网格 或 达到目标利润
- **止损**: 突破区间后止损离场
- **适用周期**: 1H/4H/日线（震荡市最佳）
- **核心指标**: 价格区间, 网格间距, 每层仓位
- **胜率预估**: 60%-80%（单笔）
- **注意事项**: 单向趋势市持续亏损；需设硬性风控
- **参考**: Grid Trading Complete Guide

## CAT-016 斐波那契回调策略 (Fibonacci Retracement)
- **类别**: 趋势/回调
- **核心**: 趋势回调在38.2%/50%/61.8%获支撑/阻力后延续
- **入场条件**: 上升趋势回调至38.2%/50%/61.8%+反转K线→BUY；下降趋势反弹至同上位置+反转K线→SELL
- **出场条件**: 目标前高/前低 或 161.8%扩展位
- **止损**: 跌穿61.8%(做多失效)+0.5ATR
- **适用周期**: 4H/日线/周线
- **核心指标**: 斐波那契回调线, K线形态
- **胜率预估**: 45%-55%
- **注意事项**: 有自我实现性；不要单独用，需配合结构确认
- **参考**: *Fibonacci Trading: How to Master the Time and Price Advantage*

## CAT-017 期权备兑开仓策略 (Covered Call)
- **类别**: 期权收入
- **核心**: 持有标的+卖出虚值看涨期权收权利金，牺牲上行换稳定收入
- **入场条件**: 持有100股标的，卖出1手虚值Call（行权价比现价高2%-5%），期限30-45天
- **出场条件**: 期权到期或被行权(交割股票) 或 价格跌破关键支撑平仓
- **止损**: 股票跌破关键支撑位时平仓
- **适用周期**: 30-45天持有期
- **核心指标**: Delta, 隐含波动率(IV), 权利金收入
- **胜率预估**: 70%-85%（方向性胜率高，但收益有限）
- **注意事项**: 暴跌时保护有限；强势上涨时踏空
- **参考**: CBOE Covered Call Strategy Guide

## CAT-018 期权跨式/宽跨式策略 (Straddle/Strangle)
- **类别**: 波动率/事件驱动
- **核心**: 重大事件前同时买入Call+Put，赌波动爆发
- **入场条件**: 财报/非农/议息前1-3天，买入ATM Call+ATM Put(跨式) 或 OTM Call+OTM Put(宽跨式)
- **出场条件**: 事件后IV回落平仓 或 任一端达2:1盈亏比时部分平仓
- **止损**: 组合损失超过权利金的50%-70%
- **适用周期**: 事件日套利（1-5天）
- **核心指标**: 隐含波动率(IV), 历史波动率(HV), IV百分位
- **胜率预估**: 40%-55%（但盈利时盈亏比可能很高）
- **注意事项**: IV在事件前可能已被定价；Theta在事件前加速损耗
- **参考**: Natenberg, Sheldon *Option Volatility and Pricing*

## CAT-019 波动率均值回归策略 (Short Volatility)
- **类别**: 波动率/套利
- **核心**: IV通常高于实际波动率，做空波动率赚取溢价
- **入场条件**: VIX处于历史高位(>75%百分位)，卖出宽跨式 或 做空VIX期货/ETN
- **出场条件**: VIX回归均值 或 期权接近到期平仓
- **止损**: VIX突破历史极值或3倍标准差
- **适用周期**: 周线/月线
- **核心指标**: VIX, VIX期货曲线(Contango/Backwardation)
- **胜率预估**: 70%-85%
- **注意事项**: 尾部风险极大(黑天鹅)；VIX ETN有展期损耗
- **参考**: Taleb, Nassim *Dynamic Hedging*

## CAT-020 RSI超买超卖+背离策略
- **类别**: 反转/均值回归
- **核心**: RSI进入极端区域+背离确认，预示价格即将反转
- **入场条件**: RSI<30+底背离+看涨K线→BUY；RSI>70+顶背离+看跌K线→SELL
- **出场条件**: RSI返回50附近(中性区) 或 价格触及布林中轨
- **止损**: 近期波段低/高点 + 1ATR
- **适用周期**: 1H/4H/日线
- **核心指标**: RSI(14), K线形态
- **胜率预估**: 50%-65%
- **注意事项**: 强趋势中RSI可长时间停留在超买/超卖区，不要逆势接飞刀
- **参考**: Wilder, J. Welles *New Concepts in Technical Trading Systems*

## CAT-021 产业链基本面深度分析
- **类别**: 基本面/价值
- **核心**: 不靠技术指标，通过网络搜索深入分析公司的产业链地位、订单、供应商、护城河、资金实力
- **入场条件**:
  - 产业链位置强势（上下游议价权高）+ 订单增长趋势 + 行业龙头/垄断地位 + 强力资金支持 → **BUY conf≥0.70**
  - 产业链位置中等 + 订单稳定 + 有护城河但非垄断 + 正常资金支持 → **BUY conf=0.50~0.60**
  - 供应商或客户集中度过高(>50%) → conf-0.15（单一依赖风险）
  - 大股东减持或机构大幅减仓 → conf-0.20
  - 基本面极好但技术面差 → **HOLD 等入场时机**（用AGT-002/003找技术入场点）
  - 基本面恶化（订单下滑/客户流失/竞争加剧）→ **SELL conf≥0.65**
- **出场条件**: 基本面恶化信号出现（订单骤降/竞争格局恶化/大股东撤离）
- **止损**: 关键基本面假设被证伪（大客户流失/技术路线被替代）
- **适用周期**: 周线/月线（中长线持有数周至数月）
- **核心指标**: 产业链位置(话语权), 订单量/积压订单, 供应商集中度, 客户集中度, 市占率, 护城河(技术/品牌/网络/资源), 大股东背景, 机构持仓变化
- **胜率预估**: 55%-70%（基本面驱动，与纯技术策略互补）
- **注意事项**:
  - 需要 web 搜索数据，不是实时数据而是趋势判断
  - 搜索词：公司名+财报/订单/供应链/股东，公司名+产业链/竞争格局
  - 搜索词：公司名+机构持仓/增持/减持
  - 基本面分析成本高（多次web搜索），仅在投票量少时执行完整分析
  - 优先分析持仓股（NVDA/MSFT/META/GOOGL/CLSK）和拟买入标的
  - 与AGT-008主力+基本面策略的区别：CAT-021更侧重产业/供应链/资金端，AGT-008更侧重盘口/技术面+基本面交叉验证
- **参考**: 《彼得·林奇的成功投资》, Porter's Five Forces 产业链分析框架


## CAT-022 Ichimoku Cloud 趋势跟踪策略
- **类别**: 趋势跟踪/多指标
- **核心**: 利用一目均衡表的云层(Tenkan/Kijun/Senkou/Kumo)综合判断趋势方向与支撑阻力，价格突破云层且转换线>基准线时入场
- **入场条件**: (1)价格收于云层(Senkou Span A/B)上方为多头/下方为空头；(2) Tenkan-sen(转换线)上穿Kijun-sen(基准线)形成黄金交叉；(3) Chikou Span(延迟线)与价格走势无背离且位于价格上方(多头)/下方(空头)；(4) Kumo(云层)未来方向偏多(未来云层变厚且倾斜向上)
- **出场条件**: (1) Tenkan-sen下穿Kijun-sen形成死叉；(2)价格回落至云层内部且Chikou Span与价格发生背离
- **止损**: 置于云层下边缘(多头)或上边缘(空头)外加1ATR
- **适用周期**: 日线/周线
- **核心指标**: Tenkan-sen(9), Kijun-sen(26), Senkou Span A(52中线), Senkou Span B(104中线), Chikou Span(26滞后), Kumo厚度
- **胜率预估**: 45%-55%（假突破多，但盈亏比通常可达2.5:1以上）
- **注意事项**: 盘整市Kumo平坦时信号频繁失效；参数(9,26,52)适合日股/外汇，对A股可能需要调整

## CAT-023 Renko 砖图反转策略
- **类别**: 反转/趋势
- **核心**: 将价格过滤为固定砖块大小(ATR倍数)的Renko图，砖块颜色转变作为趋势反转信号，忽略时间轴和微小噪音
- **入场条件**: (1)多头：连续3根同色砖块形成趋势，第四根砖块方向与前一根相反时入场。砖块大小设为1.5倍-2倍ATR(14)。(2)双砖突破：价格形成双砖突破前高且砖块为绿色时入场做多
- **出场条件**: (1)出现相反颜色的第一根砖块即平仓；(2)砖块序列出现连续2根反向砖块确认趋势反转
- **止损**: 入场砖块的底部下方1砖块大小
- **适用周期**: 15分钟-4小时
- **核心指标**: Renko砖块(固定ATR%), ATR(14), 砖块颜色序列, 砖块计数
- **胜率预估**: 50%-60%（假信号少但入场延迟大）
- **注意事项**: 砖块大小选择至关重要；Renko无法在常规K线软件中回测；无法反映夜间跳空；不适用于横盘震荡市

## CAT-024 收益率曲线斜率宏观轮动策略
- **类别**: 宏观/大类资产轮动
- **核心**: 基于收益率曲线斜率（10年期-2年期利差）的变化方向和绝对值，在股票/债券/商品/现金之间进行资产配置轮动
- **入场条件**: (1)Bull Steepener：2Y10Y利差<0(倒挂)且走阔→做多股票/做空债券；(2)Bear Flattener：利差>100bp且收窄→做多长期国债/做空周期股；(3)倒挂解除：利差从深度倒挂快速回升→大幅超配股票
- **出场条件**: (1)利差方向性移动停止且ROC回归零轴；(2)ISM与利差方向背离

- **适用周期**: 周线/月线（持有1-6个月）
- **核心指标**: 10Y-2Y利差, 5Y-30Y利差, TIPS盈亏平衡通胀率, ISM制造业PMI, 美元指数
- **胜率预估**: 55%-65%
- **注意事项**: 利差数据在QT期间可能失真；需要跨市场交易权限；利差倒挂平均领先衰退12-18个月

## CAT-025 跨资产相关性离散策略
- **类别**: 统计套利/相关性
- **核心**: 利用资产间滚动相关性的均值回归特性，当短期相关系数显著偏离长期均值(>2σ)时，做多相关性回归或做多相关性离散
- **入场条件**: 滚动20日Pearson相关系数与200日均值偏离>2σ时建仓。例如SPY-TLT的20日相关系数跌破-0.8(历史均值为-0.4)→做空SPY+做多TLT
- **出场条件**: (1)相关性回归至长期均值±0.5σ范围内即平仓；(2)持仓超过30交易日未回归则强制平仓
- **止损**: 相关性Z-score扩大到入场时的1.5倍；或-3%总仓位
- **适用周期**: 日线持有10-60天
- **核心指标**: Pearson/Spearman滚动相关系数, 相关系数Z-score, 成分股两两相关性中位数, 隐含相关系数指数
- **胜率预估**: 60%-70%（均值回归特性稳定，但尾部风险极大）
- **注意事项**: 2008年和2020年相关性趋近于1，均值回归策略惨败；相关系数本身不稳定，需检查结构断点

## CAT-026 Gamma Scalping 策略
- **类别**: 期权/波动率
- **核心**: 买入平价附近ATM期权获得正Gamma，通过不断Delta对冲将"时间价值衰减"转化为"从价格波动中获利"，核心是赚取已实现波动率高于隐含波动率的部分
- **入场条件**: (1)买入跨式/宽跨式，剩余期限14-45天；(2)IV处于历史低位(百分位<30%)，预期大幅波动；(3)有明确催化剂事件但方向不明；(4)Delta初始对冲至接近零
- **出场条件**: (1)已实现波动率开始低于IV时清仓；(2)Gamma收益达到IV水平的2倍时了结；(3)距离到期<5天平仓；(4)事件后IV塌陷立即平仓
- **止损**: Theta亏损累计超过入场权利金的50%；或IV上升但标的未动
- **适用周期**: 日内-数日（高频对冲，日内需多次调整Delta）
- **核心指标**: Gamma, Delta, Theta, Vega, 已实现波动率, IV及百分位, Gamma/Theta比率
- **胜率预估**: 单笔35%-45%但盈利通常大于亏损
- **注意事项**: 交易成本极高；gamma临近到期爆炸性放大；最容易低估"波动率陷阱"(IV高但HV低)

## CAT-027 并购套利策略
- **类别**: 事件驱动/套利
- **核心**: 公告并购后做多目标公司股票、做空收购方股票（现金收购只做多目标），赚取收购价与市价的价差
- **入场条件**: (1)spread大于历史可比交易中位数的1.5倍；(2)现金收购或全股票收购（需对冲）；(3)监管路径清晰；(4)双方董事会批准，无竞购方；(5)交易规模>1亿美元
- **出场条件**: (1)交易完成日结算退出；(2)spread收窄至年化<3%时提前平仓；(3)任何负面信号立即平仓

- **适用周期**: 1-6个月
- **核心指标**: spread百分比, 年化spread收益率, 交易完成概率, 监管审查阶段

- **注意事项**: 最大风险是交易延迟而非失败；必须在公告后立刻行动；不能做空小盘股（融券成本高）；单笔<5%总资金

## CAT-028 季节日历效应策略
- **类别**: 季节性/统计套利
- **核心**: 利用资产价格在特定日历日期(月份/周/节假日/财报季)的可重复统计异常进行定向押注，本质是市场微观结构和投资者行为偏差
- **入场条件**: (1)月度效应：月末最后1日+月初前3日买入(Turn-of-month效应)；(2)周内效应：周五尾盘买入、周一开盘卖出；(3)节假日效应：主要节假日前2日买入，节后第2日卖出
- **出场条件**: 固定日历时点出场(T+N日)；若该年该期效应未出现则放弃
- **止损**: 入场后-2%绝对止损
- **适用周期**: 持有2-5天
- **核心指标**: 日历效应历史收益率统计(过去15年), 月份平均收益率, 周一/周五年化收益差异, Santa Rally历史数据
- **胜率预估**: Turn-of-month约65%-75%；Santa Rally约70%-80%
- **注意事项**: alpha逐年衰减；交易成本/滑点对高频率策略侵蚀严重；需要大量历史数据验证统计显著性(至少15-20年)；黑天鹅年全部失效

## CAT-029 HMM 隐马尔可夫模型择时策略
- **类别**: AI/量化/择时
- **核心**: 使用隐马尔可夫模型(HMM)将市场分为3-4个隐藏状态(牛市/熊市/震荡/高波动)，只在模型后验概率>0.7且ROC确认时交易，不同状态切换不同参数
- **入场条件**: (1)HMM从熊市/震荡状态切换到牛市状态且后验概率>0.7；(2)20日累计回报率>0%确认；(3)状态标签持续至少3根K线
- **出场条件**: (1)HMM牛市状态后验概率降至0.5以下；(2)状态翻转到熊市或高波动；(3)20EMA被向下突破
- **止损**: ATR(14)×2，价格朝有利方向+2R后移至保本
- **适用周期**: 日线（1H可加速状态切换）
- **核心指标**: HMM模型(高斯发射, 3-4状态), 20日对数收益, 20日已实现波动率, 20日成交量变化, 状态转移概率矩阵
- **胜率预估**: 45%-55%（胜率较低但R:R通常>2.5:1）
- **注意事项**: HMM对回看窗口选择敏感(建议252-504个交易日)；必须加入状态持续性过滤器(最少3根K线)避免频繁切换；每63个交易日重新训练防止模型漂移；结合贝叶斯先验从更长历史稳定估计

## CAT-030 多因子Z-Score合成排名策略
- **类别**: 因子投资(价值+质量+低波)
- **核心**: 对全市场股票按价值(PE/PB/PFCF)、质量(ROE/负债率/毛利率稳定性)和低波(60M beta/60M日收益标准差)的合成Z-score排名，做多前10%、做空后10%，月度再平衡
- **入场条件**: (1)月末收盘再平衡，计算6个因子的Z-score；(2)合成Z-score=等权平均；做多前10%、做空后10%；(3)最低市值>10亿美元；最低20日成交量>500万美元；(4)排除金融和公用事业

- **胜率预估**: 55%-65%（多头60%-70%，空头45%-55%）

- **入场条件**: (1)每月再平衡：按央行政策利率排名G10货币，做多前3(等权)、做空后3(等权)；(2)仅当多空利差>200bp时入场；(3)两个篮子20日MA斜率均为正(动量过滤)

## CAT-035 市场轮廓价值区间交易 (Market Profile / 拍卖理论)
- **类别**: 微观结构/成交量
- **核心**: 从"过量"(低成交量尾部)均值回归至价值区间(VA, 包含70%TPO成交量的价格带)，以及在价值区间外出现接受(高成交量新价位)时顺势突破交易
- **入场条件**: 均值回归：价格超出VAH/VAL>1.5x价值区间宽度+低成交量(<20日平均)→做回POC方向。趋势延续：价格在VA外收盘+高成交量接受(非尾部)→做突破方向
- **出场条件**: 均值回归：在POC或VAH/VAL出场。趋势延续：20EMA跟踪，价格重新进入前日VA时退出
- **止损**: 均值回归：超过过量尾部+0.5ATR。趋势延续：突破K线低点-0.5ATR
- **适用周期**: 日内（30min/1H，每Session构建TPO）
- **核心指标**: TPO计数, POC(最高TPO点数), VAH(70%成交量的上界), VAL(下界), 价值区间宽度, 跨日VA重叠
- **胜率预估**: 55%-65%(均值回归尾部)/50%-60%(趋势延续)
- **注意事项**: 至少需要3天日内TPO数据做背景；新间驱动的跳空日不适用；早盘和尾盘的Profile行为不同；使用TPO(时间)而非仅Volume Profile；配合市场Delta确认订单流

## CAT-036 聪明钱订单流与暗池分析策略
- **类别**: 微观结构/大单
- **核心**: 追踪累计Delta背离、大额交易(>10,000股或>20万美元)检测、暗池成交量比飙升，在价格变动前识别机构吸筹/派发
- **入场条件**: 做多：(a)累计Delta背离：价格低点更低但累计Delta低点更高；(b)近10分钟>3笔大单在bid/mid；(c)暗池量比>20日均值1.5x；(d)Time & Sales显示吸收(卖盘不跌价)。做空：反向条件
- **出场条件**: (1)累计Delta衰竭(价格涨但Delta跌)；(2)暗池比回复均值；(3)跟踪止损20EMA或ATR(14)×2
- **止损**: 最近摆动低点下方0.5xATR(做多)/摆动高点上方0.5xATR(做空)
- **适用周期**: 日内（1min/5min执行，15min/1H方向偏见）
- **核心指标**: 累计Delta(买卖量差), 暗池量比(暗池/总成交量), 大单计数(>10K股), Time & Sales, VWAP, 买卖价差压缩
- **胜率预估**: 55%-65%（需要合理的成交量读盘技能）
- **注意事项**: 需要Level 2/NYSE TAQ或暗池数据源——价格昂贵；延迟关键——Tick级信号几分钟内衰减；暗池数据质量因供应商而异；必须结合价格结构(支撑/阻力)使用；最适合巨量大盘股(SPY/QQQ/AAPL/MSFT/NVDA)

## CAT-037 ETF套利策略 (NAV-市价偏差回归)
- **类别**: 统计套利/ETF
- **核心**: 利用ETF市价与成分股加权净值(NAV)之间的瞬时价差，当偏差超出交易成本+缓冲带时做多低估端+做空高估端，套取回归收益
- **入场条件**: (1)ETF市价与IOPV(实时NAV)偏差>0.3%(小盘ETF放宽至0.5%)；(2)偏差由流动性短暂失衡导致(成交单笔大单)+非基本面事件驱动；(3)做多低估端/做空高估端，构成市场中性配对；(4)偏差在2分钟内保持稳定(非报价错误)；(5)仅交易流动性前50的ETF(SPY/QQQ/IWM/XLF等)
- **出场条件**: (1)偏差收敛至0.1%以内时平仓；(2)最大持有5分钟，超时强制平仓；(3)或偏差突然扩大>0.5%时止损
- **止损**: 偏差超过入场值的1.5倍时止损；或个股出现停牌/临时交易暂停
- **适用周期**: 1分钟量级，持有1-5分钟（高频套利）
- **核心指标**: ETF市价/IOPV偏差(基点), 偏差波动率(1分钟滚动), ETF成交量(1分钟), 成分股交易状态, 盘口深度
- **胜率预估**: 65%-75%（高频，单笔薄利但高频率）
- **注意事项**: 需要直连交易所数据(L2行情)和极低延迟基础设施；交易成本是最大侵蚀——佣金和滑点可能吃掉大部分alpha；仅在市场正常时段操作(9:45-15:45 ET)；在ETF重平衡日偏差扩大但回归更快——机会更多但风险更高

## CAT-038 可转换债券套利策略 (Convertible Arbitrage)
- **类别**: 可转债/市场中性
- **核心**: 买入可转债+做空对应股票Delta份数，对冲方向性风险，赚取可转债隐含波动率被低估的收益和信用利差收益
- **入场条件**: (1)筛选Delta中性对冲后名义收益率>5%或年化>10%的可转债；(2)Delta中性比率在0.6-0.9之间(偏股型可转债有足够Gamma)；(3)信用评级不低于BBB-，剩余期限>1年；(4)隐含波动率低于正股已实现波动率(60日)的80%；(5)发行规模>1亿美元，日均成交量>500万美元
- **出场条件**: (1)可转债到期/赎回/回售触发时；(2)Delta偏离>0.15后重新对冲成本过高时平仓；(3)信用评级下调至BB+以下时立即平仓
- **止损**: 可转债价格跌破面值(100)时减仓50%，跌破90时清仓；套利组合最大亏损-3%
- **适用周期**: 持有1-6个月
- **核心指标**: 可转债隐含波动率, 正股60日HC, Delta, Gamma, 信用利差(CDS/OAS), 转换溢价, 到期收益率(YTM)
- **胜率预估**: 70%-80%（市场中性，波动率低，但流动性风险存在）
- **注意事项**: 2008年可转债套利基金几乎全军覆没——流动性同时枯竭且股票对冲失效；信用风险是最大尾部因素；Delta需每日调整(高频但成本可控)；需要可转债定价模型(Black-Scholes修正版)；最佳实施角度是全对冲基金而非单笔交易

## CAT-039 期权波动率偏斜交易策略 (Skew Trading)
- **类别**: 期权波动率
- **核心**: 利用虚值看跌期权(OTM Put)与虚值看涨期权(OTM Call)之间隐含波动率的相对定价偏差——当偏斜指数(SKEW)极端高或低时，做多相对便宜的腿、做空相对贵的腿
- **入场条件**: (1)CBOE SKEW指数>150(市场极度恐慌)时：卖出OTM Put价差(卖出较低行权价Put、买入更低Put做保护)、买入OTM Call价差——赌恐慌消退后偏斜回复正常；(2)SKEW<115(市场过度自满)时：买入OTM Put·卖出OTM Call——赌尾部风险被低估；(3)具体执行用SPX的30-45天期权链，价差在10-20点左右；(4)入场时净Theta为正(时间价值为你工作)
- **出场条件**: (1)SKEW回归至120-140的正常区间时平仓；(2)或达到2:1盈亏比时部分止盈(50%)，剩余仓位置放弃剩余到期；(3)或距到期<7天时平仓
- **止损**: 组合亏损超过入场权益金的50%；或SKEW继续朝不利方向移动超过入场时10个点
- **适用周期**: 30-45天持有（SKEW回复通常需2-4周）
- **核心指标**: CBOE SKEW指数, SPX各期限(7D/30D/60D)的25-delta Put IV - 25-delta Call IV价差, VIX期限结构斜率
- **胜率预估**: 60%-70%（偏斜回复特性较稳定，但在尾部事件中可不只回复而持续扩大）
- **注意事项**: SKEW极端值不常见(一年几次)——信号少但质量高；2008年SKEW冲至170+，做空偏斜的策略惨败；流动性注意——远OTM期权价差宽；偏斜从极端回复通常发生在2-4周内而非1-2天；用价差(非裸期权)限制风险最重要

## CAT-040 指数再平衡效应交易策略 (Index Rebalancing Front-Run)
- **类别**: 事件驱动/量化
- **核心**: 在指数(S&P500/Russell2000/NASDAQ 100)季度/年度再平衡前，提前判断哪些股票将被调入/调出，在公告后、生效前做多调入股/做空调出股，赚取被动基金调仓的被迫买卖压力
- **入场条件**: (1)提前2周筛选：对比指数规则(市值/流动性/行业)与当前成分股，预测调入/调出候选；(2)在正式公告后第2个交易日建仓(规避首日的短线投机波动)；(3)调入做多：预计新增指数化资金流入=调入股日均量×估计跟踪资金倍数(通常3-5倍日均量)；(4)调出做空：预计流出压力相同
- **出场条件**: (1)生效日(T日)收盘前平仓50%；(2)T+1日收盘前平仓剩余50%——大多数再平衡效应在T+1消退；(3)或盘中出现反常量(提前建仓的机构抢跑)时立即平仓
- **止损**: 调入股-3%(预期之外没有进入指数)；调出股+3%(没有预期之外的保留)；或指数委员会临时变更规则
- **适用周期**: 再平衡公告日至生效日后1日，约5-10个交易日
- **核心指标**: 指数再平衡日历(S&P/Russell/Nasdaq), 日均成交量, 估计跟踪资金量, 调入/调出概率预测模型, 公告日前后成交量/波动飙升
- **胜率预估**: 65%-80%（再平衡效应有充分的学术和实践证据，但alpha逐年下降）
- **注意事项**: alpha逐年下降(越来越多量化基金提前交易S&P再平衡)；Russell再平衡规模最大(S&P追踪资金更多但更透明)；流动性和冲击成本是关键——低流动性小盘调入股可能冲击成本吃掉收益；做空调出股的融券成本要考虑；指数委员会有临时豁免或调整规则的权利(不可预测风险)

## CAT-041 VIX期货期限结构展期收益策略 (Contango Roll Yield)
- **类别**: 波动率期货/展期
- **核心**: 在VIX期货期限结构处于Contango(远期>近期)时，做多VIX期货次月合约+每月滚动，赚取展期收益(contango的天然衰减)，同时限定期权对冲尾部跳升
- **入场条件**: (1)VIX期货期限结构处于Contango(前月VX1<次月VX2<三月VX3)，且contango斜率(月差)>0.5点；(2)VIX水平<25(恐慌期contango转backwardation后该策略失效)；(3)做多VX2(次月合约)，每月滚动；持仓加OTM VIX call作为尾部对冲(买入VX3的25-delta call保护)；(4)仓位规模限制(每笔<组合风险预算的0.5%)
- **出场条件**: (1)每月展期日：卖出VX2、买入新的VX2；(2)VIX期限结构转为Backwardation(VX1>VX2)时立即平仓；(3)或VIX突破35时平仓(恐慌爆发)
- **止损**: 组合止损-5%；或单月展期亏损超过预算
- **适用周期**: 月度滚动(持有25-30天)
- **核心指标**: VX1, VX2, VX3期货价格, 滚动价差(VX2-VX1), VIX水平, VIx期限结构斜率(前月/次月/三月)
- **胜率预估**: 60%-70%（因为多时间contango，但backwardation期间剧烈亏损）
- **注意事项**: 该策略在VIX backwaardation期间(市场恐慌)大幅亏损——2008年亏损超-80%；2020年3月同样暴跌；contango的展期收益虽稳定但掩盖了真实的尾部风险；使用VIX期货ETF(如VXX/VXZ)效果差(管理费用+滚动损耗不精确)；最佳实现方式是直接交易VIX期货或在离岸期货账户操作

## CAT-042 CFTC持仓报告交易策略 (Commitments of Traders)
- **类别**: 仓位分析/逆向交易
- **核心**: 根据CFTC COT周报中商业(对冲者)与非商业(投机者/基金)的净持仓变化进行逆向交易——当投机者净多头/空头处于极端(+/-2σ)时，市场大概率反转方向
- **入场条件**: (1)选择流动性好的期货品种(标普500 e-mini/黄金/原油/10Y国债/欧元/日元)；(2)计算非商业持仓的净多/空头Z-score(基于52周滚动均值与标准差)；(3)做空信号：非商业净多头Z-score>+2(投机者极度看多)+商业空头持仓在历史高位——投机泡沫跟随；(4)做多信号：非商业净空头Z-score<-2(投机者极度看空)+商业多头在历史高位——恐慌过度；(5)入场时间：每周COT数据发布后(周五16:30 ET)的下周一开始操作
- **出场条件**: (1)Z-score回归至0附近(±0.5σ)时平仓；(2)或Z-score继续扩大到极端+3以上(确认趋势延续)时止损；(3)或4周内未回归则强制平仓
- **止损**: Z-score继续向极端方向扩大超过入场时的1.5倍；或价格趋势与持仓方向一致但连续2根周线反转确认
- **适用周期**: 周线（持有1-4周，COT周度更新后调整）
- **核心指标**: 非商业净持仓(期货+期权), 商业净持仓, 非商业净持仓Z-score(52W), 商业/非商业持仓比, 未平仓合约总量变化
- **胜率预估**: 55%-65%（极端仓位反转具有统计显著性，但时间不确定）
- **注意事项**: COT数据滞后3-4天——有时反转已经发生；参数极端值定义(2σ vs 1.5σ)直接影响信号频率和胜率；仅适用于期货不适用于股票(美股没有类似COT报告)；2020年后散户通过ETF参与度增加使COT信号复杂化；最好与其他趋势指标配合使用(如ADX>25时COT信号更可靠)

## CAT-043 跳空缺口回补交易策略 (Gap Fill)
- **类别**: 价格行为/均值回归
- **核心**: 当股票/指数在开盘产生跳空缺口(价格高于前日最高价或低于前日最低价)时，根据缺口类型(突破/普通/衰竭/中继)判断回补概率，在开盘后交易缺口的回补
- **入场条件**: (1)仅交易普通缺口(普通缺口：非财报/非宏观事件驱动、非突破关键点位)；(2)开盘前15分钟确认缺口范围：Gap Up=开盘价高于前日最高价，Gap Down=开盘价低于前日最低价;(3)Gap Up做空：开盘后价格冲高无力(15分钟未创新高)，成交量低于前日平均前15分钟的量——赌缺口回补；(4)Gap Down做多：开盘后价格下探无力(15分钟未创新低)，成交量萎缩——赌缺口回补；(5)最大缺口幅度参考：过去20日平均真实范围ATR(14)的50%-100%的缺口回补概率最高；超过2xATR的缺口很少完全回补
- **出场条件**: (1)缺口完全回补(价格填满跳空范围)时平仓；(2)或持有至收盘(日内仅做日内缺口，隔夜缺口隔天回补概率显著降低)；(3)如果价格朝缺口方向继续大幅度推进(价格远离缺口10%以上)时第一时间止损
- **止损**: 开盘后缺口的反方向移动超过缺口大小的50%(Gap Up做空后价格继续上涨超过了缺口宽度一半)；或ATR(14)×0.5
- **适用周期**: 日内（5min/15min确认，持有至收盘）
- **核心指标**: 跳空缺口大小(绝对值和ATR比率), 前15分钟成交量/20日均量比, 前15分钟价格区间(是否创新高/新低), 缺口距离关键点位(支撑/阻力/均线)的距离
- **胜率预估**: 普通缺口回补概率约70%-80%；突破缺口仅30%-45%；中继缺口约50%-60%
- **注意事项**: 必须区分缺口类型——普通缺口回补概率最高、突破缺口不回补趋势确认；趋势强市(Gap Up不回补连续上行)做空缺口会被轧空；财报/宏观新闻驱动的大缺口趋势延续概率高不赌回补；1xATR以下的小缺口回补最快；开盘前15分钟的Price Action是最重要的确认信号——放量缺口不赌回补、缩量缺口做回补方向
