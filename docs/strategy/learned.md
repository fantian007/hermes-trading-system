# AGT-003 RSI 策略学习笔记

## 2026-05-24 — RSI 周末巡检经验

### 知识点
1. **周末行为**: 美股周六/周日休市，longbridge 不提供实时行情，巡检只需确认状态即可
2. **股池为空时的策略**: sentiment-pool --list 返回空时跳过分析，无需硬性刷新
3. **周五缓存有效性**: 周五收盘数据的 RSI 在周末仍有效，但超过 2 天即为过期数据，周一开盘后需重新计算
4. **RSI 评估规则**:
   - RSI > 70 = 超买（可能回调），不触发 BUY 但需监控
   - RSI < 30 = 超卖（可能反弹），潜在 BUY 信号
   - RSI 45-60 = 中性区，无操作
   - 趋势中的超买（如强势上攻中 RSI=78）不视为卖出信号
5. **巡检输出格式**: 经验文档记录包括环境状态、操作记录、结论、下一步

# AGT-004 布林带策略学习笔记

## 2026-05-24 — 第1轮上线

### 技术要点
1. **数据获取**: longbridge kline CLI 直接获取日线数据，无需走 data-agent
2. **布林带(20,2)计算**: 
   - SMA = 20日收盘价均值
   - 上轨 = SMA + 2σ
   - 下轨 = SMA - 2σ
   - 带宽(BW) = 上轨 - 下轨
   - 价格百分比 = (价格 - 下轨) / BW × 100%
3. **SQUEEZE检测**: 当前带宽 / 50周期带宽 < 0.5 视为SQUEEZE
4. **信号定义**: SQUEEZE + 近上轨(≥85%) = 偏多突破；SQUEEZE + 近下轨(≤15%) = 偏空突破

### 信号解读原则
- 价格触及上下轨不一定是反转信号
- 带宽收缩(SQUEEZE)后的突破更有意义
- ARM.US在120.8%位置（上轨外）说明强势突破，但回调风险大
- RDDT.US/UBER.US在下轨外，可能超卖反弹
- QQQ/SOXX/XLK squeeze + 近上轨，下周一最值得关注的标的
