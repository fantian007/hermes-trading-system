# AGT-004 布林带策略学习笔记

## 2026-05-24 — 第1轮上线

### 技术要点
1. **数据获取**: longbridge kline CLI 直接获取日线数据，无需走 data-agent
2. **布林带(20,2)计算**: 
   - SMA = 20日收盘价均值
   - 上轨 = SMA + 2σ
   - 下轨 = SMA - 2σ
   - 带宽(BW) = 上轨 - 下轨
   - 价格百分比 = (价格 - 下轨) / BW × 100%
3. **SQUEEZE检测**: 当前带宽 / 50周期带宽 < 0.5 视为SQUEEZE
4. **信号定义**: SQUEEZE + 近上轨(≥85%) = 偏多突破；SQUEEZE + 近下轨(≤15%) = 偏空突破

### 信号解读原则
- 价格触及上下轨不一定是反转信号
- 带宽收缩(SQUEEZE)后的突破更有意义
- ARM.US在120.8%位置（上轨外）说明强势突破，但回调风险大
- RDDT.US/UBER.US在下轨外，可能超卖反弹
- QQQ/SOXX/XLK squeeze + 近上轨，下周一最值得关注的标的
