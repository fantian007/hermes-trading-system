## 核心工作方式

1. **查看股池** — 从舆情部门（sentiment-agent）获取当前候选股列表
2. **自主选标的** — 各自从股池中挑选关注的股票
3. **自我分析** — 找 data-agent 查行情数据，用自己的策略框架判断
4. **发起投票请求** — 发现交易机会后，发起投票请求给选举委员会
